# VDI 桌面管理系统 API 分析报告

## 目录结构概览

```
consolidator/
├── base/                      # 基础模块
├── auth/                      # 认证授权模块
├── apply_approve/             # 申请审批模块
├── citrix/                    # Citrix 集成模块
├── citrix_policy/             # Citrix 策略模块
├── component_version/         # 组件版本管理模块
├── desktop_service_management/# 桌面服务管理模块
├── file_share/                # 文件共享模块
├── guest/                     # 虚机代理模块
├── module_custom/             # 模块自定义配置
├── policy_group/              # 策略组模块
├── security_policy/           # 安全策略模块
├── snapshot/                  # 快照管理模块
├── system/                    # 系统管理模块
├── system_custom/             # 系统自定义配置
├── terminal/                  # 终端管理模块
├── user/                      # 用户管理模块
├── workflow/                  # 工作流模块
├── zone/                      # 区域管理模块
├── request_builder.py         # 主请求构建器
├── request_checker.py         # 主请求检查器
└── api_acl.py                 # API 访问控制
```

---

## 模块详细分析

### 1. base 模块 (基础模块)
**用途:** 提供请求检查器和构建器的基类实现

**主要文件:**
- `base_request_checker.py` - 请求检查器基类
- `base_request_builder.py` - 请求构建器基类

---

### 2. auth 模块 (认证授权模块)
**用途:** 管理认证服务、用户、组织单元、用户组和 RADIUS 服务

**API 数量:** 42 个

**主要功能:**
- 认证服务管理（创建、查询、修改、删除）
- 认证用户管理
- 组织单元（OU）管理
- 用户组管理
- RADIUS 服务管理
- 密码提示问题管理

---

### 3. apply_approve 模块 (申请审批模块)
**用途:** 管理资源申请组、审批组和桌面申请工单

**API 数量:** 19 个

**主要功能:**
- 资源申请组管理
- 资源审批组管理
- 桌面申请工单管理
- 申请审批流程处理

---

### 4. citrix 模块
**用途:** 管理 Citrix 交付组、计算机目录和计算机资源

**API 数量:** 17 个

**主要功能:**
- 交付组管理
- 计算机目录管理
- 计算机管理
- 用户与交付组关联

---

### 5. citrix_policy 模块
**用途:** 管理 Citrix 策略及其过滤规则

**API 数量:** 13 个

**主要功能:**
- Citrix 策略管理
- 策略项配置
- 策略过滤器管理

---

### 6. component_version 模块 (组件版本管理)
**用途:** 管理系统组件版本和升级

**API 数量:** 3 个

**主要功能:**
- 查询组件版本
- 更新组件版本
- 执行组件升级

---

### 7. desktop_service_management 模块 (桌面服务管理)

### 8. file_share 模块 (文件共享)
